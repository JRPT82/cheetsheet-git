Mis primeros pasos con GIT

GIT init
inicializar repositorio con GIT
GIT status
muestra el estado del directorio de trabajo y del área de entorno de ensayo
GIT add + 'archivos separados por espacios' o add --all
añade archivo al seguimiento
GIT commit -m
confirmacion de guardado

Subir repositorio

GIT remote add nombre rama
añadir repositorio remoto a nuestro repositorio local
GIT push nombre rama
subir archivo
